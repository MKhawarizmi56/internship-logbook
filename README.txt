MY LOGBOOK SITE - FINAL POLISHED VERSION

Main file:
- index.html

Important folders:
- assets/css/styles.css      = all design and responsive styling
- assets/js/data.js          = daily logbook entries only
- assets/js/app.js           = interaction, filters, games, guide, wallpaper, export functions
- images/                    = profile and daily evidence images
- videos/                    = welcome video
- docs/                      = helper template

How to update daily entries:
1. Open assets/js/data.js
2. Copy one entry object
3. Paste before the closing ];
4. Update date, week, day, title, tasks, tools, skills, challenge, solution, reflection, status, image
5. Save and upload to GitHub

Image naming suggestion:
- profile.jpg
- day1.jpg, day2.jpg, day3.jpg, etc.

Welcome video:
- Put your welcome video file as videos/welcome.mp4
- If no video is added, the site will show a clean placeholder instead of breaking.

GitHub Pages upload:
Upload the full folder contents:
- index.html
- assets/
- images/
- videos/
- docs/

Do not upload index.html only, otherwise CSS, JS, images and videos will not work.
